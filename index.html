Features

Load Shapefile

The Shapefile format is now a common format for storing GIS data. Thie tool allows to open this file format; view its vectors and properties.

Visualization and Identify

Visualize the Shapefile vectors and list its properties information. Identify any feature by click on map or select one row in the table.

Export to Another Format

Export Shapefile to GeoJson or CSV format for easier human reading or make it compantible with the other front-end libraries development.

SDK and CLI

Built above Node.JS libraries and components. Any feature you interested in this tool is also an SDK to use for free. The CLI is free as well on NPM.

Cross-Platform

Built with Electron. With the power of Electron, we could use Javascript to build Cross-Platform UI applications for macOS, Windows or Linux.

Open Source and Free

Shapefile Viewer is totally free and open source project. If you like to add more function, feel free to create a pull request or contact me directly.
